Тонкая прослойка к API геокодера Яндекс.Карт для .NET 4+. Написана на F#, совместимость с C# пока не протестирована.

Зависимости
====
Fsharp.Core
ServiceStack.Text

Цели
====
Результат запроса к геокодеру Яндекса возвращает массу информации, и текущий формат ответа не вполне соответствует официальной документации. Эта прослойка позволяет упростить разбор результата, возвращаемого геокодером.
